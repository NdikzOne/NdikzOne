## 👋 About Me

I'm based in Demak, Central Java, currently spending most of my time learning full-stack web development — from crafting interfaces with **React** to building APIs with **Laravel** and **PHP**. I'm still early in the journey, and I'm okay with that. Every project teaches me something new about how the frontend and backend actually talk to each other.

Right now I'm working through:
